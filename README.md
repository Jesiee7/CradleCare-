# CradleCare-
物联网嵌入式项目
该系统通过集成鸿蒙Hi3861传感器和执行器，实现对婴儿床内环境的实时监测和智能调控。同时，配合手机App和网页web端的用户界面，让家长可以随时随地监测婴儿的睡眠状态，远程控制床内环境，并获取个性化的育儿建议和服务。实现了与嵌入式硬件系统的无缝对接，通过 RESTful API 和 WebSocket 实现实时数据传输和交互。<br>
